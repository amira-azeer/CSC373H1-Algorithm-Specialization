# Stanford Algorithms Specialization – Coursera

This repository contains the solutions and implementations for assignments from the **Stanford Algorithms Specialization** offered on Coursera.

## 📚 About the Course

The specialization, taught by Stanford University, covers fundamental concepts in algorithms, including:

* Divide and Conquer
* Sorting and Searching
* Graph Algorithms
* Greedy Algorithms
* Dynamic Programming

## 🛠️ Contents

This repository includes:

* Programming assignments
* Algorithm implementations

Most solutions are implemented in **JavaScript**.
